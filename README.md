## 📖 RCT

这是一项**RCT经典文献**的复现，内容是针对肯尼亚Busia地区开展的一项驱虫随机实验项目，主要涉及其中的随机化验证部分。


👉 来源:  
原文献：https://www.socialscienceregistry.org/trials/1081

原始数据与代码：https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi%3A10.7910/DVN/28038



复现代码在Code文件夹下，其中`HW1_Table1_ed.do`为原文`Table 1`的复现代码，而`HW1_Table1_output_ed.do`是为了便于导出而重写的支持结果直接导出为xlsx文件的do文件
`HW1_table_5.do`为基于原文`Table 5`脚注的复现代码，但由于原文实际处理方式和脚注有出入，因此附上基于实际处理方式的复现代码`HW1_Table5_original.do`。



## 📖 Match

这是一项**SCM**的复现，内容是针对德州房屋净值贷款对劳动力参与率影响的一篇SCM研究文献，benchmark是DID,之后是SCM ，有趣的地方在于用了一些朴素的机器学习方法用于拟合


👉 来源:  
原文献：[https://www.socialscienceregistry.org/trials/1081](https://www.aeaweb.org/articles?id=10.1257/pol.20200683)

原始数据与代码：https://www.openicpsr.org/openicpsr/project/192289/version/V1/view
